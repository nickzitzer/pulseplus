# Backend Development Standards

<Callout type="warning" emoji="⚠️">
  These standards are mandatory for all backend development. Exceptions require architectural review approval.
</Callout>

## Core Principles

- **Security First**: Protect user data above all else
- **Observability**: Everything must be measurable
- **Consistency**: Uniform patterns across services
- **Documentation**: Code must match docs
