# JSDoc Documentation

This page provides access to our JSDoc-generated code documentation. JSDoc is a documentation generator for JavaScript that generates detailed API documentation from code comments.

<Callout type="info">
  The JSDoc documentation will open in a new tab. It provides a comprehensive view of our codebase's functions, classes, and modules.
</Callout>

## Features of JSDoc Documentation

- **Comprehensive API Reference**: Detailed documentation of all public APIs
- **Type Information**: Parameter and return type information
- **Code Examples**: Usage examples for complex functions
- **Class Hierarchies**: Visual representation of class inheritance
- **Module Organization**: Organized view of code modules and namespaces

## Generating JSDoc Documentation

To generate the JSDoc documentation, run the following command:

```bash
npm run docs:jsdoc
```

This will create the JSDoc documentation in the `documentation/nextra/public/jsdoc-static` directory.
