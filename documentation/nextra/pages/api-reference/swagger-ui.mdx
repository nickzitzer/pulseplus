This page provides access to our interactive API documentation powered by Swagger UI. The Swagger UI allows you to explore and test our REST API endpoints directly from your browser.

## Using Swagger UI

### Authentication

To test authenticated endpoints:

1. Click the "Authorize" button at the top of the Swagger UI
2. Enter your API key or JWT token
3. Click "Authorize" to save your credentials
4. Now you can test authenticated endpoints

### Testing Endpoints

To test an endpoint:

1. Click on the endpoint you want to test
2. Click the "Try it out" button
3. Fill in any required parameters
4. Click "Execute" to send the request
5. View the response below

### Understanding Responses

The response section includes:

- **Status code**: The HTTP status code of the response
- **Response headers**: Headers returned by the API
- **Response body**: The data returned by the API
- **Curl command**: The equivalent curl command for the request
