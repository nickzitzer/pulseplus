# Observability Standards

## Logging Format

```javascript
// Mandatory log structure
{
  "timestamp": "2023-01-01T00:00:00Z",
  "level": "info",
  "message": "User created",
  "service": "backend",
  "version": "1.2.3",
  "traceId": "trace_123",
  "userId": "usr_123",
  "durationMs": 150
}
```
