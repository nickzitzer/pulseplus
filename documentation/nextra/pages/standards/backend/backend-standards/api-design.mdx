# API Design Standards

## Response Formats

```json
// Success Response
{
  "success": true,
  "data": {
    "user": {
      "id": "usr_123",
      "name": "John Doe"
    }
  },
  "meta": {
    "timestamp": 1689984000,
    "traceId": "trace_123"
  }
}

// Error Response
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid email format",
    "details": [
      {
        "field": "email",
        "message": "Must be a valid email address"
      }
    ]
  }
}
```
